# Spotify-Music-Player-Clone

A simple clone of the Spotify music player interface built using HTML/CSS, JavaScript.

## Features
- Play/Pause music
- Display song list
- Skip to next/previous track
- Volume control
- Responsive UI

## Tech Stack
- HTML, CSS, JavaScript
